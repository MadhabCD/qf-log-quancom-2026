# Data Dictionary

This file defines the QF-LOG features and labels used in the study.

## Features

qber: Quantum bit error rate for one generated frame.

photon_count: Observed detector click count for one generated frame.

latency_ms: Simulated controller and processing latency in milliseconds.

abort_flag: Binary abort indicator generated from observed QF-LOG values.

## Labels

normal: Reference simulated operating condition.

partial_intercept_resend: Simulated partial intercept-resend condition.

detector_blind: Simulated detector-response suppression condition.

fiber_tap: Simulated optical attenuation condition.
